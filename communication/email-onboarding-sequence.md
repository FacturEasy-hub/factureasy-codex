## EMAIL 3 — J+3 · Email éducatif — erreurs Chorus Pro

**Objet A :** 3 erreurs courantes sur Chorus Pro (et comment les éviter)
**Objet B :** Pourquoi vos factures Chorus Pro sont parfois rejetées — et comment corriger ça

---

Bonjour **{{ prénom }}**,

90 % des factures rejetées par Chorus Pro le sont à cause des mêmes erreurs. Voici lesquelles — et comment FacturEasy les élimine automatiquement.

---

### Erreur n°1 — SIRET incorrect ou code service manquant

Certaines administrations (ministères, hôpitaux CHU, grandes collectivités) ont un **code service** (ou code UNL) spécifique que vous devez renseigner en plus du SIRET. Si vous ne l'indiquez pas, la facture est rejetée automatiquement.

**Avec FacturEasy :** notre base de données intègre les codes service des 50 000 entités publiques les plus fréquentes. FacturEasy vous alerte si un code est manquant avant l'envoi.

---

### Erreur n°2 — Numéro de marché absent

Si vous êtes sous marché public, le numéro de marché est obligatoire dans la facture. L'omettre = rejet garanti.

**Avec FacturEasy :** un champ dédié "Numéro de marché" apparaît automatiquement dès que votre destinataire est une entité soumise à la commande publique.

---

### Erreur n°3 — Montant TTC incohérent avec le taux de TVA

Une erreur d'arrondi (par exemple, saisir 19,6 % au lieu de 20 %) suffit à générer un rejet comptable chez certaines entités.

**Avec FacturEasy :** le calcul TVA est automatique. Vous saisissez le HT, le TTC se calcule sans risque d'erreur.
