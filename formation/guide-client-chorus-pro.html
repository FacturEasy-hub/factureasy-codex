Étape 1 · 30 minutes

Créer votre compte Portail Public de Facturation (Chorus Pro)

ℹ️
Vous avez peut-être déjà un compte Si vous avez des clients publics (mairies, hôpitaux, collectivités), vérifiez d'abord sur chorus-pro.gouv.fr avec votre email professionnel avant de créer un nouveau compte.

📋 Prérequis — préparez ces éléments avant de commencer

  • Votre numéro SIRET (14 chiffres) — vérifiable sur annuaire-entreprises.data.gouv.fr
  • Un email professionnel valide (celui que vous utilisez pour votre activité)
  • Votre numéro de TVA intracommunautaire (format FR + 11 chiffres)
  • Vos coordonnées d'entreprise complètes (adresse, téléphone)

📝 Étapes de création — procédure complète

1
Aller sur chorus-pro.gouv.fr
Cliquez sur "Créer un compte" en haut à droite. Vous serez redirigé vers ProConnect (l'identité numérique officielle de l'État).
2
S'authentifier via ProConnect
Utilisez votre email professionnel. Si vous n'avez pas encore de compte ProConnect, créez-en un en 5 minutes. Votre email est votre identifiant permanent.
3
Renseigner votre SIRET
Le système vérifie votre SIRET automatiquement et pré-remplit le nom de votre entreprise. Vérifiez que le nom affiché correspond bien à votre structure.
4
Compléter votre profil
Ajoutez votre numéro de TVA, vos coordonnées complètes, et désignez un administrateur (généralement vous). Renseignez également votre RIB si vous voulez recevoir des virements directs.
5
Confirmer par email
Vous recevez un email de confirmation dans les 24h. Cliquez sur le lien de validation. Votre compte est actif — vous pouvez maintenant émettre des factures.
⚠️
Attention : délai de validation La validation complète du compte peut prendre jusqu'à 48h ouvrées pour les nouvelles entreprises. Ne démarrez cette étape que si vous avez le temps d'attendre avant d'émettre votre première facture.
✅
Succès — Votre compte est actif quand : Vous pouvez vous connecter sur chorus-pro.gouv.fr, voir votre tableau de bord "Espace Fournisseur" et votre SIRET apparaît dans le menu. Passez à l'étape 2.

Étape 2 · 20 minutes

Connecter FacturEasy à Chorus Pro

💡
Ce que fait cette connexion Une fois FacturEasy connecté à votre compte Chorus Pro, vos factures seront transmises automatiquement — plus besoin de vous connecter sur deux plateformes différentes.

🔑 Générer vos clés API dans Chorus Pro

1
Accéder à la gestion des API
Dans Chorus Pro, allez dans Mon compte → Paramètres techniques → Gestion des API. Vous verrez la liste de vos applications connectées (vide au départ).
2
Créer une nouvelle application
Cliquez "Nouvelle application" → Nom : FacturEasy → Type : Logiciel de facturation → Droits : cochez Dépôt de factures et Consultation de statuts.
3
Copier votre clé API
Chorus Pro génère une Clé API (longue chaîne alphanumérique) et un Secret. Copiez-les immédiatement — le secret n'est affiché qu'une seule fois.
app.factureasy.fr/settings/integrations/chorus-pro

⚙️ Finaliser la configuration dans FacturEasy

4
Ouvrir FacturEasy → Paramètres → Intégrations
Dans le menu latéral, allez dans Paramètres → Intégrations → Cliquez sur la tuile "Chorus Pro / PPF".
5
Coller votre SIRET et vos clés API
Collez le SIRET (sans espaces), la Clé API et le Secret copiés depuis Chorus Pro. Vérifiez qu'il n'y a pas d'espace parasite avant/après les clés.
6
Tester la connexion
Cliquez "Tester la connexion". FacturEasy envoie une requête de vérification à Chorus Pro. Si la connexion est réussie, vous verrez "✅ Connexion vérifiée" en vert.
⚠️
Erreur courante : "Clé API invalide" Vérifiez que vous avez copié la clé en entier (60+ caractères). Les clés Chorus Pro commencent par cp_live_ en production ou cp_test_ en environnement de test. Assurez-vous d'utiliser la clé de production.
✅
Succès — Connexion établie quand : FacturEasy affiche "Connecté à Chorus Pro" avec votre SIRET et le nom de votre entreprise. Un indicateur vert est visible sur le tableau de bord. Passez à l'étape 3.

Étape 3 · 15 minutes

Émettre votre première facture électronique

📋 Tutoriel pas-à-pas — de zéro à la facture envoyée

1
Nouvelle facture → Choisir le destinataire
Dans FacturEasy, cliquez + Nouvelle facture. Renseignez le SIRET complet à 14 chiffres du destinataire (pas le SIREN à 9 chiffres). FacturEasy vérifie automatiquement si le destinataire est inscrit sur le PPF.
2
Renseigner l'en-tête obligatoire
Numéro de facture (format libre, doit être unique et séquentiel), date d'émission, date d'échéance. Pour les marchés publics : numéro d'engagement fourni par votre client.
3
Ajouter les lignes de prestation
Pour chaque ligne : description, quantité, prix unitaire HT, taux de TVA (20%, 10%, 5.5% ou 0%). Vous pouvez avoir plusieurs taux de TVA sur la même facture. Le total TTC se calcule automatiquement.
4
Prévisualiser et vérifier
Cliquez "Prévisualiser". Vérifiez : tous les montants sont corrects, les deux SIRET sont affichés, le numéro de facture est unique. Si tout est bon, fermez l'aperçu.
5
Émettre la facture
Cliquez "Émettre vers Chorus Pro". FacturEasy génère le fichier Factur-X et le transmet automatiquement. Vous recevez un numéro de dépôt Chorus Pro — conservez-le, c'est votre preuve d'envoi.
✅
Succès — Facture bien envoyée quand : FacturEasy affiche un numéro de dépôt Chorus Pro (format : CP-XXXXXXXXXX). La facture passe au statut "EMISE" dans votre tableau de bord. Vous recevez également un email de confirmation.
⚠️
Erreur courante 1 : "SIRET destinataire non trouvé" Vérifiez le SIRET sur annuaire-entreprises.data.gouv.fr. Si le destinataire est un établissement public, utilisez le SIRET de l'établissement concerné (pas celui du siège). Certaines collectivités ont un SIRET par service.
⚠️
Erreur courante 2 : "Numéro d'engagement manquant" (marchés publics uniquement) Le numéro d'engagement est fourni par votre client public lors de la signature du bon de commande. Si vous ne l'avez pas, appelez votre interlocuteur — cette information est indispensable. Sans elle, la facture sera rejetée automatiquement.
⚠️
Erreur courante 3 : "Numéro de facture déjà utilisé" Chaque numéro de facture doit être unique. Si vous avez annulé une facture, ne réutilisez pas son numéro. Ajoutez une lettre (ex : FE-001 → FE-001B) ou créez un nouveau numéro séquentiel.

FAQ — 10 questions fréquentes

Qu'est-ce que le PPF / Chorus Pro exactement ? ▾
Le PPF (Portail Public de Facturation), accessible via chorus-pro.gouv.fr, est la plateforme gratuite de l'État qui centralise toutes les factures électroniques B2B françaises à partir de septembre 2026. Il est géré par l'AIFE (Agence pour l'Informatique Financière de l'État). C'est votre intermédiaire obligatoire pour émettre des factures à des clients publics, et le canal légal pour les factures B2B privées dès la réforme.
Dois-je obligatoirement passer par une PDP privée payante ? ▾
Non. Le PPF (Chorus Pro) est gratuit et suffit pour être conforme à la réforme. Les PDP privées (Pennylane, Docaposte, Generix...) offrent des services supplémentaires comme l'archivage longue durée ou l'intégration comptable avancée, mais ne sont pas obligatoires pour la grande majorité des PME. FacturEasy se connecte directement au PPF.
Qu'est-ce qu'un numéro d'engagement et où le trouver ? ▾
Pour les marchés publics, chaque bon de commande est associé à un numéro d'engagement dans le système comptable de l'administration. Ce numéro doit figurer sur votre facture, sinon elle est rejetée automatiquement. Demandez-le à votre interlocuteur public (service commandes ou comptabilité) dès la signature du bon de commande — ne l'oubliez pas à ce moment-là.
Ma facture a été rejetée — quelle est la procédure exacte ? ▾
1. Consultez le motif du rejet dans FacturEasy (Dashboard → Facture concernée → "Détail du rejet").
2. Identifiez le champ problématique (SIRET, numéro d'engagement, format de montant...).
3. Créez une nouvelle facture avec le même contenu et le numéro corrigé (ex : FE-001 → FE-001A).
4. Ne tentez pas de "modifier" la facture rejetée — elle est définitivement close dans Chorus Pro.
5. Émettez la nouvelle facture. Durée moyenne de correction : 5 minutes.
Quelle est la différence entre SIRET (14 chiffres) et SIREN (9 chiffres) ? ▾
Le SIREN est le numéro d'identification de l'entreprise (9 chiffres). Le SIRET est le numéro de l'établissement spécifique (14 chiffres = SIREN + NIC sur 5 chiffres). Chorus Pro exige le SIRET complet du destinataire. Pour les clients publics qui ont plusieurs établissements (ex : une ville avec plusieurs services), vérifiez que vous utilisez le bon SIRET d'établissement sur annuaire-entreprises.data.gouv.fr.
Puis-je encore envoyer des factures PDF par email après septembre 2026 ? ▾
Non, pour les factures B2B à partir de septembre 2026. Le PDF simple par email ne sera plus accepté comme facture légalement valide entre professionnels assujettis à la TVA. La facture doit obligatoirement transiter par le PPF ou une PDP agréée. Vos clients professionnels auront le droit de refuser les PDFs, et vous vous exposez à une amende de 15€ par facture non conforme.
Que signifie le statut "Mise à disposition" et dois-je m'inquiéter ? ▾
"Mise à disposition" signifie que votre facture a bien été reçue par votre client mais qu'il ne l'a pas encore traitée (acceptée ou rejetée). C'est un statut normal — ne relancez pas avant la fin du délai légal de traitement (30 jours pour les entreprises privées, 30 jours pour les administrations). Une relance prématurée n'accélère pas le processus.
Comment émettre une facture d'avoir (note de crédit) ? ▾
Dans FacturEasy, créez une nouvelle facture et sélectionnez le type "Avoir" ou "Note de crédit". L'avoir doit référencer le numéro de la facture originale dans le champ prévu. Saisissez les montants en positif (FacturEasy gère l'inversion automatiquement). L'avoir est transmis à Chorus Pro comme n'importe quelle facture.
Combien de temps dois-je conserver mes factures électroniques ? ▾
10 ans, conformément à l'article L.123-22 du Code de commerce. Une facture électronique conforme (Factur-X, UBL, ou émise via Chorus Pro) a la même valeur légale qu'une facture papier. FacturEasy archive automatiquement toutes vos factures sur ses serveurs sécurisés. Vous pouvez également exporter et stocker vos factures localement ou sur votre comptable.
FacturEasy a-t-il accès à mes données de facturation ? ▾
FacturEasy traite vos factures pour les transmettre à Chorus Pro — c'est le fonctionnement technique du service. Vos données sont hébergées sur des serveurs sécurisés en France (conformité RGPD). FacturEasy ne partage pas vos données avec des tiers. Vos identifiants Chorus Pro (clés API) sont chiffrés et ne sont jamais accessibles par l'équipe FacturEasy. Consultez notre politique de confidentialité sur factureasy.fr/confidentialite.
